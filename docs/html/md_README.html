Descrição do Projeto

PS2 é um formato bancário estruturado usado para débitos diretos, composto por registos padronizados que garantem a validação, integridade e processamento automático das transações financeiras. O principal objetivo desse projeto é gerarmos esse tipo de arquivo através de dados guardados em ficheiros e depois analisarmos os dados em uma Dashboard.

Para a realização e cumprimento dos objetivos foram utilizadas as seguintes linguagens de programção:

  • C: Ler ficheiros e gerar o arquivo PS2 estruturado.
  • Python: Validação e manipulação de dados e visualização de métricas.

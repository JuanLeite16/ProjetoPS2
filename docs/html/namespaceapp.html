Descrição detalhada

Aplicação web interativa (dashboard) para análise de ficheiros PS2.

Implementa uma interface gráfica com Shiny for Python que permite carregar ficheiros PS2, filtrar por período e cliente, visualizar KPIs, gráficos e tabelas detalhadas dos movimentos e validações.

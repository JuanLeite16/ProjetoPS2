Funções de validação e formatação de dados.

Contém funções para validação de NIF, NIB e ficheiros PS2, bem como utilitários de formatação e acesso seguro a dados.

validations.obter_valor_seguro_df (   df,
  coluna,
  linha = 0,
  valor_padrao = "—"

Obtém um valor de um DataFrame de forma segura.

Tenta aceder ao valor da coluna e linha indicadas num DataFrame. Caso a coluna ou a linha não existam, devolve um valor padrão.

Parâmetros
dfDataFrame de onde o valor será obtido.
colunaNome da coluna a aceder.
linhaÍndice da linha (por defeito 0).
valor_padraoValor devolvido em caso de erro.
Retorna
Valor encontrado no DataFrame ou o valor padrão.

validations.validar_existencia_dados (   estrutura_dados)

◆ validar_nib()

validations.validar_nib (   nib)

Valida um NIB (Portugal) pelo algoritmo módulo 97.

Verifica se o NIB contém 21 dígitos numéricos e valida os dígitos de controlo através do cálculo módulo 97.

Parâmetros
nibString com o NIB a validar.
Retorna
True se o NIB for válido, False caso contrário.

◆ validar_nif()

validations.validar_nif (   nif)

Valida um NIF (Portugal) pelo algoritmo do dígito de controlo (módulo 11).

Verifica se o NIF tem 9 dígitos numéricos e calcula o dígito de controlo para confirmar se o último dígito é válido.

Parâmetros
nifString com o NIF a validar.
Retorna
True se o NIF for válido, False caso contrário.

◆ validar_ps2()

validations.validar_ps2 (   ficheiro)

Valida um ficheiro PS2.

Verifica se o caminho fornecido corresponde a um ficheiro existente, regular, com extensão .ps2 e não vazio.

Parâmetros
ficheiroCaminho para o ficheiro PS2 a validar.
Retorna
Objeto Path correspondente ao ficheiro validado. @raises ValueError Se o ficheiro não for válido ou estiver vazio. @raises FileNotFoundError Se o ficheiro não existir.

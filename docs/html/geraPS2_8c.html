Gera um ficheiro PS2 para um mês/ano com base nas cobranças.

Cria o ficheiro em "data/DD_AAAA_MM.ps2", escreve cabeçalho, registos e rodapé, e atualiza o cabeçalho com os totais calculados.

Parâmetros
mesMês de referência.
anoAno de referência.
n_cobrancasNúmero de cobranças a escrever.
CobrancasArray de cobranças.
Retorna
int 1 se o ficheiro foi gerado com sucesso, 0 em caso de erro.

Descrição detalhada

Módulo de utilitários genéricos.

Contém funções auxiliares para formatação de moeda, manipulação segura dos dicionários e conversão das strings numéricas.

Documentação das funções

◆ cent_to_euros()

utils.cent_to_euros (   cent)

Converte um valor em cêntimos para euros.

Divide o valor em cêntimos por 100 e formata com duas casas decimais.

Parâmetros
centValor em cêntimos.
Retorna
String com o valor convertido para euros. @raises Exception Se ocorrer erro na conversão.

◆ format_euro()

utils.format_euro (   x)

Formata um valor numérico para o formato monetário em euros.

Converte o valor para float e formata com duas casas decimais, usando vírgula como separador decimal e ponto para milhares.

Parâmetros
xValor a formatar.
Retorna
String formatada em euros ou o valor original em caso de erro.

◆ format_float()

utils.format_float (   x)

Normaliza um valor numérico em formato string.

Parâmetros
xValor a normalizar.
Retorna

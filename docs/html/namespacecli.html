Descrição detalhada

Interface de linha de comandos (CLI) para análise e geração de ficheiros PS2.

Recebe ficheiros PS2 via argumentos, processa-os e disponibiliza um menu interativo para visualizar resumos, adicionar/remover ficheiros e gerar um PS2 via executável.
